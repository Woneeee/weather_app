# git 사용법

cmd 아니면 git bash에서

1. git init
   =>깃 초기화 (저장소 생성)

2. git config --global user.name "Woneeee"
   =>내가 누군지 알려주기

3. git config ==global user.emil "a6885088@naver.com"
   =>내가 누군지 알려주기2

4. git config --list
   =>리스트로 아이디, 이메일 등록되었는지 확인

5. git remote add arigin 저장소url
   =>저장소 추가

6. git add .
   =>변경, 추가, 삭제 수정 내역 등 파일을 추가
   =>스테이징 단계까지 완료

7. git
